## Getting Started
 
To run locally, just open the file in your browser — no build step or dependencies needed:
 
```bash
git clone https://github.com/Komal123Kushwaha/portfolio.git
cd portfolio
open index.html
```
 
---
 
## Contact Form Setup
 
The contact form uses [Formspree](https://formspree.io). The form endpoint is already configured. To use your own:
 
1. Sign up at formspree.io
2. Create a new form and copy your endpoint ID
3. In `index.html`, replace the URL in `submitForm()`:
```js
   fetch('https://formspree.io/f/YOUR_FORM_ID', ...)
```
